<h1 data-importer="text" align="left">RSL-AI-Education-K12-Automation</h1>

###

<h2 data-importer="text" align="left">Descripción</h2>

###

<p data-importer="text" align="left">Repositorio oficial que contiene los scripts de Python utilizados para el cribado, filtrado y validación de la Revisión Sistemática de la Literatura titulada: "Aprendizaje adaptativo y simulación con inteligencia artificial en educación básica".</p>

###

<h2 data-importer="text" align="left">Autores</h2>

###

<p data-importer="text" align="left">- Cruzado Ventocilla, John Peter (U21313454)<br><br>- Romisoncco Almonacid, Gabriel Alessandro (U22222875)</p>

###

<h2 data-importer="text" align="left">Flujo de trabajo</h2>

###

<p data-importer="text" align="left">El proceso se automatizó mediante cuatro módulos principales:</p>

###

<p data-importer="text" align="left">1. filtrar.py: Limpieza de registros duplicados (DOI/Títulos).<br><br>2. filtro_poblacion.py: Depuración demográfica (enfocada exclusivamente en K-12).<br><br>3. filtro_ingenieria.py: Cribado técnico para aislar literatura de ingeniería/IA.<br><br>4. filtro_maestro.py: Evaluación semántica ponderada para selección de candidatos.</p>
