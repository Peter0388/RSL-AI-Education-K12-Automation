RSL-AI-Education-K12-Automation
Descripción
Repositorio oficial que contiene los scripts de Python utilizados para el cribado, filtrado y validación de la Revisión Sistemática de la Literatura titulada: "Aprendizaje adaptativo y simulación con inteligencia artificial en educación básica".

Autores
Cruzado Ventocilla, John Peter (U21313454)

Romisoncco Almonacid, Gabriel Alessandro (U22222875)

Flujo de trabajo (Pipeline)
El proceso se automatizó mediante cuatro módulos principales:

filtrar.py: Limpieza de registros duplicados (DOI/Títulos).

filtro_poblacion.py: Depuración demográfica (enfocada exclusivamente en K-12).

filtro_ingenieria.py: Cribado técnico para aislar literatura de ingeniería/IA.

filtro_maestro.py: Evaluación semántica ponderada para selección de candidatos.
